About This Web App

This Smart Irrigation System web app helps farmers decide when and how much to irrigate based on soil moisture, crop type, soil type, and days since planting.

It aims to prevent over-irrigation and under-irrigation, improving crop yield and saving water.

Farmers can follow simple instructions to measure soil moisture using household tools and calculate the precise amount of water required.

The system is built with simplicity and practicality in mind, ensuring small-scale farmers can easily use it.
